# Brick Changer Game
 - course project for Java Lab
 - 2D console game

# Tools used for building this Game
 - Java
 - Java Swing
 - IntelliJ

# How to install ?
 - Download the repo as a zip file or clone it
 - Set up Java Swing on your editor
 - Run the file "Main.java"
 - Enjoy the Game
